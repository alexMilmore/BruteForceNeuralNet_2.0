Specify architectures

Input the directory path that the images are inside

Instructions

Architectures are specified using a few rules.
_ symbols seperate layers
, symbols seperate settings
multiple different settings may be used with the | charachter

Layer settings
Each layer has 3 settings, the type of layer, settings based on the layer type and the activation function.

Single layer

conv2D, 3, relu
This specifies a convolutional 2D layer, with 3 filters using the relu activation function.
Settings for layers
currently conv2D and dense are supported
conv2D; The number determins the number of filters dense; The number determines the number of nodes in the dense layer

Specifying multiple parameters


The point of this program is to test many different neural nets
conv2D, 3|4, relu
tests both a [conv2D, 3, relu] and a [conv2D, 4, relu] layer autonomously.
This allows many different archetypes to be tested with ease.

Final example

Bringing this all together we can specify a few architectures in one command
conv2D,4|8|16,relu|tanh_conv2D,8|16|32,relu|tanh_dense,64|128|256,relu|tanh_dense,2,sigmoid
This will test 144 different architectures, all with just one string.
